# f-Socials — User Documentation

**f-Socials is a lens, not a judge.** Paste a YouTube link, an article URL, or a transcript, and f-Socials shows you *how that content is built* — the claims it makes, the evidence behind each one, the persuasion techniques in its framing, and other credible angles on the same topic. It gives you the tools to decide what to think. It never tells you a video is "true" or "false," and it never puts a reliability score on a person or channel.

## The one-paragraph version

Misleading content rarely lies outright — it *frames*. It picks which facts to show, names a villain, borrows the authority of a "study," and leans on your emotions. f-Socials slows that down. It pulls out each factual claim and tries to find real evidence for it (and says so honestly when it can't), it names the framing techniques and shows you the exact quote that triggered each one, and it points you to other credible coverage of the same topic. The goal isn't to think *for* you — it's to make the machinery visible so you can think for yourself.

## The promise we hold ourselves to

Three commitments are enforced in the software itself, not just stated here:

1. **No verdicts on content, no labels on people.** Source-reliability tiers attach to *sources and citations only* — never to a creator, channel, or author.
2. **Honest absence.** When there's no solid external evidence for a claim, f-Socials says "no sufficient evidence found" rather than guessing. That's a feature, not a gap.
3. **Nothing ships that can't show its work.** A report is only marked ready if every claim's stated evidence strength is backed by a citation it can actually point to, and every framing flag has a real quote behind it.

See the in-app **Methodology page** (the "How f-Socials works" link in any report's footer) for the live, detailed version of all three.
